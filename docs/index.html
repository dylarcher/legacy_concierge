Buttons

Live <ui-button> components with interactive states.

Solid Buttons (Default)

Primary actions. Use color attribute for variants.

Dark Light Blue Indigo Cyan Teal Green 
<ui-button color="dark">Dark</ui-button>
<ui-button color="blue">Blue</ui-button>

Outline Buttons

Secondary actions. Add outline attribute.

Default Outline Disabled 
<ui-button outline>Default Outline</ui-button>
<ui-button outline disabled>Disabled</ui-button>

Plain Buttons

Tertiary/ghost actions. Add plain attribute.

Plain Button Disabled 
<ui-button plain>Plain Button</ui-button>

Dialogs

Live <ui-dialog> components for modal interactions.

Basic Dialog

Click the button to open a dialog. Press Escape or click outside to close.

Open Dialog Confirm Action Are you sure you want to proceed with this action? This cannot be undone. Cancel Confirm 
<ui-dialog id="my-dialog" size="md">
  <ui-dialog-title>Dialog Title</ui-dialog-title>
  <ui-dialog-description>Description text</ui-dialog-description>
  <ui-dialog-actions>
    <ui-button plain onclick="this.closest('ui-dialog').close()">Cancel</ui-button>
    <ui-button color="blue">Confirm</ui-button>
  </ui-dialog-actions>
</ui-dialog>

<!-- Open with JavaScript -->
document.getElementById('my-dialog').open();
