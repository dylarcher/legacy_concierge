Accordion

A collapsible content component with animated expand/collapse transitions. Supports independent panels (multiple open) or exclusive mode (one at a time).

Basic Accordion

What is Tailwind CSS Plus?

Tailwind CSS Plus is a collection of professionally designed UI components built with Tailwind CSS. It provides ready-to-use components that follow modern design patterns and accessibility best practices.

 How do I get started?

Simply import the components you need and use them in your HTML. Each component is self-contained and styled with Tailwind CSS classes.

 Is it accessible?

Yes! All components are built with accessibility in mind, including proper ARIA attributes, keyboard navigation, and focus management.

Usage

<!-- Import the accordion component -->
<script type="module">
  import "../blocks/modules/accordion.js";
</script>

<!-- Basic accordion (multiple panels can be open) -->
<ui-accordion>
  <ui-accordion-panel open>
    <ui-accordion-header>Section Title</ui-accordion-header>
    <ui-accordion-content>
      <p>Panel content goes here...</p>
    </ui-accordion-content>
  </ui-accordion-panel>
  <ui-accordion-panel>
    <ui-accordion-header>Another Section</ui-accordion-header>
    <ui-accordion-content>
      <p>More content...</p>
    </ui-accordion-content>
  </ui-accordion-panel>
</ui-accordion>

<!-- Exclusive mode (only one panel open at a time) -->
<ui-accordion exclusive>
  ...
</ui-accordion>

<ui-accordion> Methods

expandAll()Opens all panels (non-exclusive only)
collapseAll()Closes all panels

<ui-accordion-panel> Methods

open()Opens the panel
close()Closes the panel
toggle()Toggles open/closed state

Events

EventDetailDescription
accordion-change { panel, open } Fired when any panel state changes
accordion-expand { panel } Fired when a panel expands
accordion-collapse { panel } Fired when a panel collapses

JavaScript API

// Get accordion element
const accordion = document.querySelector("ui-accordion");

// Listen for events
accordion.addEventListener("accordion-change", (event) => {
  console.log("Panel changed:", event.detail.panel);
  console.log("Is open:", event.detail.open);
});

// Programmatic control
accordion.expandAll();   // Open all panels (non-exclusive mode)
accordion.collapseAll(); // Close all panels

// Control individual panels
const panel = accordion.querySelector("ui-accordion-panel");
panel.open();   // Open this panel
panel.close();  // Close this panel
panel.toggle(); // Toggle state

// Check panel state
console.log(panel.isOpen);     // boolean
console.log(panel.isDisabled); // boolean
