Scroll To Top Button

A floating button that appears when the user scrolls up past a threshold. Clicking it smoothly scrolls the page back to the top.

Live Demo

Scroll down this page, then scroll back up to see the button appear in the bottom-right corner. The button is already active on this page.

Usage

<!-- Import the component -->
<script type="module">
  import "../blocks/components/buttons/button-return.js";
</script>

<!-- Basic usage (appears bottom-right after scrolling 300px) -->
<ui-scroll-to-top></ui-scroll-to-top>

<!-- Custom threshold and position -->
<ui-scroll-to-top
  threshold="500"
  position="bottom-left"
  label="Back to top">
</ui-scroll-to-top>

Attributes

AttributeTypeDefault
thresholdnumber300
position"bottom-right" | "bottom-left""bottom-right"
labelstring"Scroll to top"

Events

EventDescription
scroll-to-topFired when button is clicked

Behavior

  • Button only appears when scrolling up and past the threshold
  • Automatically hides when scrolling down or near the top
  • Uses smooth scroll animation with CSS transitions
  • Fixed positioning with z-index 50 for overlay

Extender Card

Factory function that creates expandable cards using the native <details> element. Supports toggle mode (expand/collapse) or link mode (static card with CTA link).

Toggle Mode (click to expand)

Link Mode (static card with CTA)

Usage

import { createExtenderCard, createExtenderCardList, createExtenderCardText }
  from "../blocks/components/card-details.js";

// Toggle mode with list content
const card = createExtenderCard({
  title: "Locations",
  buttonText: "View Locations",
  image: "/images/beach.jpg",
  mode: "toggle",
  body: createExtenderCardList([
    "Los Angeles",
    "San Diego",
    "Ventura"
  ])
});

// Toggle mode with text content
const textCard = createExtenderCard({
  title: "About",
  buttonText: "Read More",
  image: "/images/about.jpg",
  mode: "toggle",
  body: createExtenderCardText("Description text here...")
});

// Link mode (no expand, just a link)
const linkCard = createExtenderCard({
  title: "Services",
  buttonText: "Learn More",
  image: "/images/services.jpg",
  mode: "link",
  href: "/services"
});

document.body.appendChild(card);

createExtenderCard() Options

OptionTypeDefault
titlestring"Card Title"
buttonTextstring"View Location"
imagestring""
imageAltstring"Card background"
bodystring | HTMLElement | HTMLElement[]null
mode"toggle" | "link""toggle"
hrefstring"#"
openbooleanfalse

Helper Functions

createExtenderCardList(items)Creates a styled bullet list
createExtenderCardText(text)Creates a styled paragraph

Exported Style Constants

These can be imported for custom styling:

  • EXTENDER_CARD_BASE - Base container styles
  • EXTENDER_CARD_COLLAPSED / EXTENDER_CARD_EXPANDED - Height states
  • EXTENDER_CARD_TITLE / EXTENDER_CARD_BODY - Typography
  • EXTENDER_CARD_BUTTON / EXTENDER_CARD_CLOSE_BUTTON - Button styles

Global Footer

A full-featured footer Web Component with link columns, newsletter signup form, social links, and legal/copyright section. Supports slots for customization.

Live Preview

Note: Preview shows the default template with placeholder links.

Usage

<!-- Import the component -->
<script type="module">
  import "../blocks/components/global-footer.js";
</script>

<!-- Basic usage with default template -->
<global-footer company="My Company"></global-footer>

<!-- Custom copyright year -->
<global-footer
  company="Acme Inc"
  copyright="2024">
</global-footer>

<!-- With custom sections using slots -->
<global-footer company="Acme Inc">
  <content-info slot="links" heading="Products">
    <a href="/product-a">Product A</a>
    <a href="/product-b">Product B</a>
  </content-info>
  <content-info slot="links" heading="Resources">
    <a href="/docs">Documentation</a>
    <a href="/support">Support</a>
  </content-info>
</global-footer>

Global Navbar

A responsive navigation bar Web Component with dropdown menus, mobile drawer, search dialog, and full keyboard navigation support.

Live Preview

Try resizing the iframe or opening in a new tab to see mobile menu behavior.

Usage

<!-- Import the component -->
<script type="module">
  import "../blocks/components/global-navbar.js";
</script>

<!-- Basic usage with nav links as children -->
<global-nav brand="My Company">
  <a href="/">Home</a>
  <a href="/about">About</a>
  <a href="/services">Services</a>
  <a href="/contact">Contact</a>
</global-nav>

<!-- With custom logo -->
<global-nav
  logo="/logo.svg"
  logo-dark="/logo-dark.svg"
  logo-alt="Company Logo"
  brand="Acme">
  ...
</global-nav>

<!-- With CTA button -->
<global-nav
  brand="Acme"
  cta-text="Sign Up"
  cta-href="/signup">
  ...
</global-nav>

<!-- Fixed positioning (default) vs transparent -->
<global-nav fixed transparent brand="Acme">
  ...
</global-nav>

Attributes

AttributeTypeDefault
brandstring"Legacy Concierge"
logostringBuilt-in SVG
logo-darkstringSame as logo
logo-altstringBrand name
cta-textstringundefined
cta-hrefstringundefined
fixedbooleantrue
transparentbooleanfalse

Events

EventDescription

Methods

closeMenu()Closes mobile menu
toggleMenu()Toggles mobile menu

Slots

Features

  • Responsive dropdown menus with viewport-aware positioning
  • Mobile drawer menu with focus trap and backdrop
  • Search dialog modal with keyboard shortcuts
  • Escape key closes menus and dialogs
  • ARIA attributes for accessibility
  • Automatic resize handling for open dropdowns

Hero Banner

A full-height hero Web Component with video or image background, heading, description, CTA buttons, scroll indicator, and optional audio controls.

Live Preview (with image background)

Usage

<!-- Import the component -->
<script type="module">
  import "../blocks/components/region-hero.js";
</script>

<!-- Basic usage with image background -->
<hero-banner
  no-video
  heading="Your health, Our Purpose."
  description="Exceptional care with compassion."
  primary-cta="Get Started"
  primary-href="/signup"
  image="/images/hero.jpg"
  image-alt="Hero background">
</hero-banner>

<!-- With video background -->
<hero-banner
  heading="Welcome"
  video="/videos/hero.mp4"
  video-type="video/mp4"
  image="/images/poster.jpg">
</hero-banner>

<!-- Text alignment -->
<hero-banner
  align="center"
  heading="Centered Hero"
  description="Content is centered on all screen sizes.">
</hero-banner>

Attributes

AttributeTypeDefault
headingstring"Your health, Our Purpose."
descriptionstringDefault description
imagestringDefault beach image
image-altstring""
videostringDefault video
video-typestring"video/mp4"
no-videobooleanfalse
primary-ctastringundefined
primary-hrefstring"#"
secondary-ctastringundefined
secondary-hrefstring"#"
align"left" | "center""left" on desktop

Constants

HERO_VIDEO_ENABLEDDefault video state (currently false)
HERO_VIDEO_DISABLEDInverse for convenience
BANNER_TEMPLATE()Returns HTML template string

Features

  • Full viewport height (min-h-dvh)
  • Video background with autoplay, loop, and muted by default
  • Image fallback for video poster and non-video mode
  • Animated scroll indicator button (bouncing chevron)
  • Audio mute/unmute toggle for video (bottom-right corner)
  • Special heading formatting: comma-separated text gets italic styling
  • Responsive text sizing with clamp()
